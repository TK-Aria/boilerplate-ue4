# ue4-boilerplate
boilerplate for unreal engine 4. 

## setup

setup

```
git init
git lfs install
git remote add origin ${git_repository_url}
git fetch
git merge origin/main
```

clone

```
git lfs clone ${git_repository_url}
```

list

```
git lfs ls-files
```
